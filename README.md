## Node

JavaScript runtime environment.

## Express

Node library.

## Passport JS

Uses the following node packages (passport and one passport strategy):

passport
passport-google-oath20


## GoogleOAuth

Google API service. 

### Flow

User goes to /auth/google to sign in with their google account.
